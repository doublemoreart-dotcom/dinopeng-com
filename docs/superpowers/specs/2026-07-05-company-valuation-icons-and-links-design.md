# AI 公司估值排行榜圖示與連結樣式設計

## 目標

改善「AI 公司估值排行榜」的品牌辨識與細節一致性，處理三項問題：公司圖示不完整、圖示與文字未對齊、詳情抽屜來源連結沿用瀏覽器預設藍色。

## 公司圖示

- 私人 AI 原生公司與上市 AI／多元科技公司共 20 家，皆配置本機公司圖示。
- 圖示優先取自公司官方網站的 favicon 或官方品牌資產，不依賴頁面執行時的外部圖示服務。
- 圖示儲存在 `assets/company-logos/`，使用可被瀏覽器直接載入的 PNG、ICO 或 SVG。
- 每家公司仍保留縮寫 fallback；只有檔案不存在或載入失敗時才顯示縮寫。
- 不為圖示加入單一品牌色背景，避免 Light／Dark Mode 下產生額外色塊衝突。

## 對齊與尺寸

- 卡片、表格與詳情抽屜沿用各自既有尺寸，但共用相同的 `inline-flex` 對齊模型。
- 表格公司欄改為明確的圖示加文字排列，圖示不以外距控制基線，文字與圖示採垂直置中。
- 詳情抽屜的圖示與「公司類型／排名、公司名稱」文字群組垂直置中。
- 圖片使用 `object-fit: contain`，並以一致內距避免品牌圖示碰觸容器邊緣。

## 連結樣式

- 詳情抽屜中的來源連結使用全站既有連結色與底線間距，不使用瀏覽器預設藍色。
- 一般狀態使用 `var(--page-text)` 或既有連結變數；hover／focus 使用 `#378ADD`。
- Light／Dark Mode 都需維持可讀性與清楚的鍵盤焦點狀態。

## 實作範圍

- 更新 `index.html` 的公司圖示映射與排行榜／抽屜 CSS。
- 將相同修改同步到目前日期版本 `ai_industry_penetration_2026-07-01.html`。
- 新增缺少的本機公司圖示資產，不移除其他頁面仍可能使用的舊圖示。
- 更新靜態契約測試，確認 20 家公司都有圖示映射、檔案存在，且抽屜連結使用統一樣式。

## 驗收標準

- 私人榜與上市榜 20 家公司均顯示品牌圖示，載入失敗時可退回縮寫。
- 卡片、表格與抽屜內的圖示與相鄰文字視覺置中。
- 抽屜來源連結在 Dark Mode 不呈現瀏覽器預設藍色。
- Light／Dark Mode、桌機與手機版皆無文字或圖示溢出。
- 完整 Node 測試通過，瀏覽器 console 無錯誤。
